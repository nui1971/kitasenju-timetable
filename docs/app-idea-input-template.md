## アプリのアイデア・使い方

どんな場面で使うか：
出かける直前に北綾瀬駅の次の電車まで何分かを確認するアプリ
| 項目 | 内容 |
|---|---|
| 誰が使うか | 自分（個人利用） |
| いつ使うか | 外出前・家を出る直前 |
| 何のために使うか | 次の列車まで何分かを素早く確認する |

一番重要なことは何か：


表示したい情報は何か：
| エリア | 内容 |
|---|---|
| ヘッダー | C20バッジ・駅名（北綾瀬/Kita-Ayase）・現在時刻・日付 |
| ダイヤバッジ | 平日／土・休日（今日に対応する方をグリーン表示） |
| フィルター | 「綾瀬行を表示しない」チェックボックス |
| 次の列車カード | 時刻・行き先・種別・あとXX分 |
| 列車リスト | 時刻・行き先・種別・あとXX分　5本表示＋さらに表示（+5本） |
- あとXX分は60分の場合はあと1時間、それ以上の場合はあとxx時間yy分と表示したい。
- 列車リストはデフォルト5本・展開で+5本（合計10本）・「追加分を非表示」で折りたたみ

操作したいことは何か：情報の表示
不要・非表示にしたいものは：

その他・気になること：
なし
